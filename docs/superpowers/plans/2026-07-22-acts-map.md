# Acts Map Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Korean Acts explorer with whole-book and Paul-journey modes, a color legend, and unmistakable selection state.

**Architecture:** Generate Acts 1–28 from the provided 개역개정 HWP source and serve map context data from an API route. A mode-aware client explorer filters a shared map dataset; a generic map component receives layer colors and selected-ID styling.

**Tech Stack:** Next.js 15, React 19, TypeScript, MapLibre GL, Vitest, 개역개정 HWP source.

## Global Constraints

- Use the provided 개역개정 HWP source for Acts text.
- Keep full-book and Paul-only modes on one `/books/acts` route.
- Show a text color legend and `✓ 선택됨` in the selected list item.
- Test API, mode filtering, and selection state before final push.
