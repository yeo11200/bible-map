# 모든 교인을 위한 시작 안내 설계

## 목적

성경 지도 탐험의 주요 대상을 특정 숙련도나 연령대가 아닌 모든 교인으로 확장한다. 성경 공부 입문자도 부담 없이 이해할 수 있는 문구를 사용하되, 모든 사용자는 하나의 로마서 탐험 경로로 진입한다.

## 사용자 흐름

홈 화면에서 사용자는 `로마서 탐험 시작`을 선택해 로마서 지도 탐험으로 이동한다.

## 화면 구성

- 히어로 영역의 문구는 성경 숙련도를 전제하지 않고, 장소·이야기·본문의 연결을 함께 살펴보는 경험으로 설명한다.
- 로마서 카드 하나만 두어 중복된 진입 선택지를 만들지 않는다.
- 지도와 구절을 탐색하는 핵심 기능은 변경하지 않는다.

## 접근성 및 반응형 원칙

- 명확한 `로마서 탐험 시작` 링크 텍스트를 사용한다.
- 색상만으로 단계나 행동을 구분하지 않는다.

## 테스트

- 홈 화면에 `처음 오셨나요?` 중복 안내가 표시되지 않는다.
- `로마서 탐험 시작`이 로마서 탐험 경로를 가리킨다.
- 기존 로마서 탐험 선택 및 구절 표시 테스트는 계속 통과한다.

## 범위 제외

- 사용자 숙련도 저장, 온보딩 완료 상태 저장, 로그인 기능은 이번 변경에 포함하지 않는다.
- 다른 성경책·인물 여정·오늘의 콘텐츠는 추가하지 않는다.
