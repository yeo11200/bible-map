# 여호수아 지도 탐험 설계

## 목표

개역개정 여호수아 1–24장을 지도 위의 장소·사건·본문으로 연결한다.

## 범위

- 요단 도하와 여리고 정복(1–8장)
- 가나안 정복과 땅의 확보(9–12장)
- 지파별 땅 분배와 세겜 언약 갱신(13–24장)
- 지도 지점 선택, 관련 구절, 해당 지점에서 읽을 장, 1–24장 전체 본문 리더
- 지도 색상 범례와 선택 지점 강조

## 데이터와 화면

`data/joshua-context.ts`는 지도 지점의 순서, 분류, 관련 장·구절을 제공한다. `data/gaeokgaejeong-joshua.json`은 사용자 제공 개역개정 HWP 원문에서 추출한 전체 본문이다. `/api/books/joshua`는 이 두 데이터를 내려주고, `/api/books/joshua/map`은 GeoJSON을 내려준다.

`JoshuaExplorer`는 API에서 받은 데이터만 사용해 지도를 렌더링한다. 지도에는 요단 도하·정복·분배/언약을 서로 다른 색으로 표기하고, 선택 지점은 더 크게 표시한다.

## 지리 원칙

성경에 등장하는 고대 지명의 현대 좌표는 확정이 어려울 수 있다. 고고학·성서지리에서 널리 쓰이는 교육용 대표 좌표를 사용하며, 불확실한 곳은 설명에 단정하지 않는다.

## 검증

- API 테스트: 1번 지점이 요단 도하이고, 1–24장이 모두 연결되며, 개역개정 본문이 내려오는지 검증한다.
- 컴포넌트 테스트: 정복/분배 모드 전환, 선택 강조, 장 이동과 실제 본문 표시를 검증한다.
- 전체 테스트와 프로덕션 빌드를 실행한다.

## 사용자 제약

이번 작업에서는 커밋이나 푸시를 수행하지 않는다. 실패 테스트를 먼저 작성하고 확인한 뒤 구현한다.
