# 사도행전 지도 탐험 설계

## 목표

사도행전 1~28장을 지도에서 읽되, 전체 구속사 흐름과 바울의 선교 여행을 각각 명확하게 탐색할 수 있게 한다.

## 탐험 모드

### 전체 여정

- 1~12장: 예루살렘 교회 탄생, 유대·사마리아 확장, 안디옥 교회
- 13~21장: 바울의 1~3차 선교 여행
- 22~28장: 예루살렘 체포, 가이사랴, 로마행과 로마 도착

### 바울의 선교 여행

- 1차 선교 여행: 13~14장
- 2차 선교 여행: 15~18장
- 3차 선교 여행: 18~21장
- 로마행: 21~28장

두 모드는 동일한 본문 데이터와 장소를 사용한다. 전체 여정은 교회 확장의 큰 흐름을, 바울 모드는 여행별 경로와 장소를 강조한다.

## 지도와 색상

- 교회 탄생·확장: 금갈색
- 바울 선교 여행: 청록색
- 로마행: 보라색
- 위치가 합의되지 않은 지점: 회색 보조색과 `추정 위치` 텍스트
- 지도 상단에 색상 견본과 의미를 함께 보여 주는 고정 범례를 제공한다.

## 선택 상태

- 선택한 목록 버튼은 진한 배경, 고대비 테두리, `✓ 선택됨` 텍스트를 표시한다.
- 선택한 지도 핀은 더 큰 반지름과 흰색 외곽선으로 표시한다.
- 선택한 지점이 속한 경로는 굵고 불투명하게, 나머지 경로는 얇고 반투명하게 표시한다.
- 지도 핀과 목록 버튼은 같은 색상 체계를 사용한다.

## 본문

- 개역개정 HWP 원문에서 사도행전 28장을 추출해 서버 API가 제공한다.
- 각 지도 지점에는 핵심 절과 관련 장 목록을 제공하고, 전체 본문 리더에서 해당 장을 읽을 수 있다.
- 로마 도착 지점은 로마서의 로마행 맥락과 연결 가능한 안정적인 ID를 사용한다.

## 검증

- API가 사도행전 28장과 두 탐험 모드 데이터를 반환하는지 검증한다.
- 모드 전환이 지도 항목을 바꾸고, 선택한 항목이 `✓ 선택됨`을 표시하는지 검증한다.
- GeoJSON에 색상 레이어와 선택 상태를 구분할 수 있는 속성이 있는지 검증한다.
